🌡️ Temperature Converter

A simple and interactive web application that allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin in real-time. Built using basic HTML, CSS, and JavaScript — perfect for beginners to understand DOM manipulation and unit conversion.

🚀 Features

Convert between Celsius, Fahrenheit, and Kelvin.
Clean, responsive UI.
Real-time conversion as the user types.
Lightweight and beginner-friendly project.

🛠️ Tech Stack

HTML5 – Structure and layout
CSS3 – Styling and responsiveness
JavaScript – Logic and interactivity
